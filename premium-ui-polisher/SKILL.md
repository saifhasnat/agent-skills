# Premium UI Polisher

This skill ensures that all web development tasks follow the highest standards of modern design and user experience. It avoids generic or "MVP-style" designs in favor of premium, state-of-the-art interfaces.

## core Principles

1.  **Wow Factor**: At first glance, the user should be impressed. Use harmonious color palettes (HSL), sleek dark modes, and modern typography.
2.  **Visual Excellence**: Avoid basic colors (plain red/blue/green). Use smooth gradients and depth.
3.  **Dynamic Feel**: Interfaces must feel "alive" with hover effects, micro-animations, and smooth transitions.
4.  **No Placeholders**: Always use real images or generate them to keep the UI feeling complete.

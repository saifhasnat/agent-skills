# Firebase Standard Manager

This skill helps maintain clean and secure Firebase projects. It provides templates and patterns for common Firebase tasks, ensuring consistency across all environments.

## Core Features

1.  **Firestore Organization**: Hierarchical data patterns for scalability.
2.  **Security First**: Templates for rules that restrict access by default.
3.  **Auth Best Practices**: Guidelines for enabling providers and managing custom claims.
4.  **Functions Integration**: Standardizing trigger handling and logging.
